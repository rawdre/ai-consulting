# Agent Workspace Notes

This folder exists so Acerbot can work inside the `ai-consulting` repo with less guesswork.

Contents:
- `repo-map.md` — high-level map of pages, assets, and workflows
- `priorities.md` — current priorities and likely next moves
- `sensitive-notes.md` — reminders about secrets/sensitive material that should not be copied into memory or git changes casually

Keep secrets out of tracked files. Store only operational notes and project understanding here.

# AI Tool School

AI Tool School is a small static website built to help beginners learn practical AI tools through hands-on guides instead of abstract theory.

Current topics:

- Codex
- Claude Code
- OpenClaw
- Notion
- Warp
- A guided "first build" experience

## What this project is for

This project is designed to prove something simple:

- a beginner can start from zero
- use AI tools in a practical way
- build something structured and useful
- and publish the result as a real website

The site is meant to feel like a starter school, not just a documentation dump.

## Main files

- `index.html`: homepage and learning hub
- `first-build.html`: guided first project experience
- `codex-best-practices.html`
- `claude-code-complete-guide.html`
- `openclaw-complete-guide.html`
- `notion-complete-guide.html`
- `warp-complete-guide.html`
- `site.css`
- `site.js`

## Features

- searchable guide library
- learning paths
- progress checklist stored in the browser
- English / Portuguese toggle on the main site shell
- beginner-first positioning and demo-friendly messaging

## Publishing

This site is intended to be placed inside the `rawdre/ai-consulting` GitHub Pages repository under a subfolder such as:

- `tool-school/`

That would make the final URL look like:

- `https://rawdre.github.io/ai-consulting/tool-school/`

## Notes

- `node_modules/` is ignored and should not be published
- this is a static site, so it can be hosted easily on GitHub Pages
- guide pages can be expanded further with deeper Portuguese translations if needed
